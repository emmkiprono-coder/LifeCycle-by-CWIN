# CWIN LifeCycle Admin

AI-powered care coordination platform with React frontend and Express.js API.

## Features

- Beautiful React homepage
- Express.js REST API
- 5 AI Agents (Appointment, Billing, Coordination, Documentation, Family)
- Demo data included
- Vercel deployment ready

## Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Auto-deploys on push
4. Frontend and API both live

## API Endpoints

- GET /api/health
- GET /api/clients
- GET /api/appointments
- GET /api/bills
- GET /api/tasks
- POST /api/ai/request

## Quick Start Local

```bash
npm install
npm start
```

API runs on http://localhost:5000

## Production

Deployed on Vercel with:
- Frontend: React (static)
- Backend: Node.js (serverless)
- Environment: Production
- Status: Active
